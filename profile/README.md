### Google Apps Script (GAS) Projects

> **Disclaimer:** [Yorsh](https://yorsh.co) is not affiliated with, endorsed by, or associated with Google LLC.

#### Modules
- [`gas-sheetdb`](https://github.com/yorsh-co/gas-sheetdb) - Database-like storage layer for Google Apps Script projects.
- [`gas-config`](https://github.com/yorsh-co/gas-config) - In-sheet configuration management for Google Apps Script projects.
- `gas-secrets` (upcoming - June 2026) - Secrets management for Google Apps Script projects.

#### Libraries

OAuth 2.0 Flows:
- `gas-auth-google` (upcoming - June 2026) - OAuth 2.0 flow and token management for Google APIs as a Google Apps Script library.
- `gas-auth-calendly` (upcoming - June 2026) - OAuth 2.0 flow and token management for the Calendly API as a Google Apps Script library.
- `gas-auth-square` (upcoming - June 2026) - OAuth 2.0 flow and token management for the Square API as a Google Apps Script library.

API Libraries:
- `gas-api-appsscript` (upcoming - July 2026) - Apps Script API wrapper as a Google Apps Script Library.
- `gas-api-calendly` (upcoming - July 2026) - Calendly API wrapper as a Google Apps Script Library.
- `gas-api-square` (upcoming - July 2026) - Square API wrapper as a Google Apps Script Library.
- `gas-api-close` (upcoming - July 2026) - Close CRM API wrapper as a Google Apps Script Library.

#### Apps
- `forms-scores-publisher` (upcoming - July 2026) - Publish accumulated Google Forms scores results to a web app.
